# JavaScript ဆိုတာဘာလဲ။

JavaScript ဆိုတာ ဝဘ်ဆိုက်တွေကို အသက်ဝင်လှုပ်ရှားပြီး Interactive ဖြစ်အောင် လုပ်ပေးတဲ့ Cross-platform, Object-oriented စာရေးနည်း (Scripting language) တစ်ခုဖြစ်ပါတယ်။ ဒါတင်မကသေးပဲ Node.js လိုမျိုး Server ဘက်မှာ သုံးရတဲ့ JavaScript ပုံစံမျိုးတွေလည်း ရှိပါသေးတယ်။ JavaScript ဟာ သူ့အမည်မှာကိုက Script ဆိုတဲ့အသုံးအနှုန်းပါနေပြီး ဖြစ်ပါတယ်။ Interpreted
Language တစ်မျိုး၊ Script Language တစ်မျိုး ဖြစ်ပါတယ်။ 

Browser ထဲမှာသုံးတဲ့ Client-side JavaScript ကတော့ Browser နဲ့ Document Object Model (DOM) ကို ထိန်းချုပ်ဖို့ Objects တွေ ပါပြီးသားပါ။ အဲ့ဒါကြောင့် Browser ထဲမှာဆိုရင် JavaScript က ဝဘ်စာမျက်နှာ (DOM) ရဲ့ ပုံစံကို ပြောင်းလဲပစ်နိုင်တာဖြစ်ပါတယ်။ JavaScript ကို စတင်တီထွင်စဉ်က အဲ့ဒီလို Client-side Language အနေနဲ့ တီထွင်ခဲ့ကြတာပါ။ 

နောက်တော့မှ Node လို့ခေါ်တဲ့ နည်းပညာတစ်မျိုး ထွက်ပေါ်လာပြီး JavaScript ကုဒ်တွေကို Browser မလိုတော့ဘဲ ဘာသာပြန် အလုပ်လုပ်ပေးနိုင်လာတာပါ။၊ Server ပေါ်က Node.js JavaScript ကလည်း Server ပေါ်မှာ ပတ်သက်တာတွေ လုပ်နိုင်ဖို့ Objects တွေ ထပ်ဖြည့်ပေးထားတဲ့အတွက် Browser ဘက်က ပို့လိုက်တဲ့ Request တွေကို Server ပေါ်မှာအအလုပ်လုပ်ပေးနိုင်ပါတယ်။

ဒါကြောင့်ကနေ့အချိန်မှာတော့ JavaScript ဟာ General Purpose ခေါ် ကဏ္ဍစုံမှာ အသုံးပြုနိုင်တဲ့ စွယ်စုံသုံး Language တစ်ခုဖြစ်နေပါပြီ။ JavaScript ကိုအသုံးပြုပြီး Web Application တွေ၊ Mobile App တွေ Desktop Software တွေရေးသားဖန်တီးလို့ ရပါတယ်။

## JavaScript နဲ့ Java

JavaScript နဲ့ Java ဟာ နာမည်အားဖြင့် ဆင်တူပေမယ့် တစ်ခုနဲ့တစ်ခု ဆက်စပ် သက်ဆိုင်ခြင်းမရှိတဲ့ သီးခြားနည်းပညာတွေပါ။ Java ကို လုပ်ငန်းသုံးဆော့ဖ်ဝဲတွေ ဖန်တီးဖို့နဲ့ Android App တွေ ဖန်တီးဖို့အပိုင်းမှာ ပိုအသုံးများကြပါတယ်။ JavaScript ကိုတော့ Web App တွေဖန်တီးဖို့နဲ့ Web Services တွေဖန်တီးဖို့ ပိုအသုံးများကြပါတယ်။ Java ဟာ Compiled Language တစ်မျိုးဖြစ်ပြီး JavaScript ဟာ Interpreted Language တစ်မျိုးပါ။ ပြီးတော့ Java ဟာ Statically Typed သဘောသဘာဝ ရှိပြီး JavaScript ကတော့ Loosely Typed သဘောသဘာဝရှိပါတယ်။ 

## Getting Started with JavaScript

JavaScirpt နဲ့ စအလုပ်လုပ်ဖို့ဆိုရင် modern browser တွေဖြစ်တဲ့ Firefox, Chrome, Microsoft Edge နဲ့ Safari တို့ထဲကတစ်ခုရှိရင်ရပါပြီ။ အဲ့ဒီ browser တွေက JavaScript Feature တွေအကုန်နီးပါးကို support ပေးပါတယ်။ Broswer တွေမှာပါတဲ့ console မှာ JavaScript code တွေကို တိုက်ရိုက် run လို့ရပါတယ်။ Right Click > Inspect (or) F12 ကိုနှိပ်ပြီး Console tab ကိုသွားရမှာဖြစ်ပါတယ်။
